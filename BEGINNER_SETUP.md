# Beginner Setup

You do not need web development for this project. This is an Android app project.

## Install Once

1. Install Android Studio from the official Android developer site.
2. Open Android Studio.
3. Let it install the Android SDK, Android Emulator, and Java 17.
4. Choose `Open` and select this folder:
   `C:\Users\trqgp\Documents\Codex\2026-05-08\can-you-build-android-app-no`
5. Wait for Gradle Sync to finish.
6. Press the green Run button.
