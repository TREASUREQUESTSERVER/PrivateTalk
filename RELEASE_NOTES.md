# PrivateTalk Test Release

## APK

Use this APK for invited tester phones:

`privatetalk-onesignal-live.apk`

## Included

- Invite-only Google sign-in.
- Firestore real-time chats.
- Username search and add.
- End-to-end encrypted text.
- Free-plan compressed image and voice-note sending.
- WebRTC call signaling and in-app audio/video calls.
- OneSignal SDK configured for push subscription.
- Admin panel hosted on Firebase Hosting.

## Free-Plan Limits

- Firebase Storage is disabled.
- Firebase Cloud Functions are disabled.
- Push delivery still needs OneSignal send workflow/backend wiring.
- Calls work best when both users are online in the app.
