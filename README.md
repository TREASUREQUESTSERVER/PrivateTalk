# PrivateTalk Android

PrivateTalk is the starter native Android app for an invite-only WhatsApp-style messenger.

This first version includes:

- Invite-only welcome flow
- Chats, status, calls, and profile tabs
- Chat room UI with messages, attachment, voice note, and call actions
- Backend planning for auth, invite codes, encrypted messaging, media, calls, and notifications

## Open It

1. Install Android Studio.
2. Open this folder as an Android project:
   `C:\Users\trqgp\Documents\Codex\2026-05-08\can-you-build-android-app-no`
3. Let Android Studio install the Android SDK and sync Gradle.
4. Run the app on an Android emulator or phone.

## Current Status

The app is a working UI starter with mock data. The next real backend step is Firebase Auth + Firestore invite validation, then encrypted one-to-one messages.
